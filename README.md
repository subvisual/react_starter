React Starter
=============

This project is a starter project to build React applications. In includes:

* React
* Redux
* React Router
* Parcel
* CSS Modules
* Eslint
* Prettier
* Stylelint

Prerequisites:

* Node
* Yarn

Installation:

* bin/setup

Development:

* bin/server

Building:

* bin/build

Generator
---------

This boilerplate contains a generator script that helps with the creation of
new components. Run `bin/gen -h` for more information.
